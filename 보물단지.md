# kingshot.jeab.dev API 문서

> **Base URL:** `https://kingshot.jeab.dev`  
> **인증:** 없음 (Public API)  
> **응답 형식:** JSON

---

## 전체 아키텍처

```
클라이언트 (Vite + React SPA)
│
├── /assets/index-bbd0dd2c.js   ← 메인 앱 번들
└── /assets/vendor-af84ddd4.js  ← 서드파티 라이브러리 (GSAP 등)

서버 (REST API)
│
├── /api/config
├── /api/stats
├── /api/stats/highlights
├── /api/players
├── /api/players/{id}
├── /api/news
├── /api/news/events
├── /api/codes
└── /avatars/{hash}.png
```

## 페이지 로드 시 호출 순서

```
1. GET /api/config
2. GET /api/stats
3. GET /api/stats/highlights
4. GET /api/news?limit=20&offset=0
5. GET /api/news/events
6. GET /api/codes
7. GET /api/players?limit=1&offset=0
8. GET /api/players?tagged=1&limit=100&offset=0
9. GET /api/players?tagged=1&limit=200&offset=100

```

## API 엔드포인트 상세

### 1. 사이트 설정

```
GET /api/config
```

**응답 예시**

```json
{
  "allKingdoms": true,
  "allowedStates": []
}
```

| 필드            | 타입     | 설명                                    |
| --------------- | -------- | --------------------------------------- |
| `allKingdoms`   | boolean  | 전체 왕국 표시 여부                     |
| `allowedStates` | number[] | 허용된 왕국 번호 목록 (비어있으면 전체) |

### 2. 전체 통계

```
GET /api/stats
```

**응답 예시**

```json
{
  "total_players": 282145,
  "redeeming_players": 2556,
  "total_redemptions": 75507,
  "dead_players": 753,
  "inactive_players": 1241
}
```

| 필드                | 타입   | 설명                       |
| ------------------- | ------ | -------------------------- |
| `total_players`     | number | 전체 트래킹 플레이어 수    |

| `inactive_players`  | number | 비활동 플레이어 수         |

### 4. 플레이어 목록

```
GET /api/players
```

**쿼리 파라미터**

| 파라미터 | 타입   | 필수 | 설명                              |
| -------- | ------ | ---- | --------------------------------- |
| `limit`  | number | ✅   | 한 번에 가져올 수 (최대 200 추정) |
| `offset` | number | ✅   | 페이지네이션 오프셋               |
| `tagged` | `1`    | ❌   | `1`이면 태그된 플레이어만 반환    |
| `search` | string | ❌   | 플레이어명 검색                   |
| `state`  | number | ❌   | 왕국 번호(kingdom ID)로 필터      |

**요청 예시**

```
GET /api/players?limit=100&offset=0
GET /api/players?tagged=1&limit=100&offset=0
GET /api/players?search=Deus&limit=20&offset=0
GET /api/players?state=893&limit=50&offset=0
```

**응답 예시**

```json
{
  "players": [
    {
      "id": "131060914",
      "username": "Tub Snus",
      "avatar_url": "/avatars/a041dd5cbf3b8b0cfd23bb1990e133ae.png",
      "state": 893,
      "sort_order": -4580,
      "last_refreshed_at": "2026-06-15T05:30:45Z",
      "town_hall_level": 55,
      "tag": "GOAT"
    }
  ],
  "total": 512
}
```

**플레이어 객체 필드**

| 필드                | 타입              | 설명                                        |
| ------------------- | ----------------- | ------------------------------------------- |
| `id`                | string            | 플레이어 고유 ID                            |
| `username`          | string            | 플레이어 닉네임                             |
| `avatar_url`        | string            | 아바타 이미지 경로 (`/avatars/{hash}.png`)  |
| `state`             | number            | 소속 왕국 번호 (kingdom ID)                 |
| `sort_order`        | number            | 정렬 기준값 (음수, 절댓값이 클수록 상위)    |
| `last_refreshed_at` | string (ISO 8601) | 마지막 데이터 동기화 시각                   |
| `town_hall_level`   | number            | 본부 레벨 (TG 레벨과 매핑: TG5 ≈ 55)        |
| `tag`               | string \| null    | 커뮤니티 태그 (미태그 플레이어는 필드 없음) |

**tag 종류**

| 태그             | 의미            |
| ---------------- | --------------- |
| `GOAT`           | 최상위 플레이어 |
| `KING` / `QUEEN` | 왕족            |
| `RAT`            | 배신자          |
| `BABE` / `BISH`  | 커뮤니티 별명   |
| `Baby`           | 신규/저레벨     |
| `BRO`            | 일반 우호       |
| `Cougar`         | 커뮤니티 별명   |
| `F2P LEGEND`     | 무과금 전설     |

### 6. 활동 뉴스 피드

```
GET /api/news
```

**쿼리 파라미터**

| 파라미터 | 타입   | 필수 | 설명                |
| -------- | ------ | ---- | ------------------- |
| `limit`  | number | ✅   | 가져올 항목 수      |
| `offset` | number | ✅   | 페이지네이션 오프셋 |

**응답 예시**

```json
[
  {
    "id": 10389,
    "kind": "name_change",
    "message": "Snus is now Tub Snus",
    "avatar_url": "/avatars/a041dd5cbf3b8b0cfd23bb1990e133ae.png",
    "old_avatar_url": "/avatars/a041dd5cbf3b8b0cfd23bb1990e133ae.png",
    "state": 893,
    "player_id": "131060914",
    "player_name": "Tub Snus",
    "old_name": "Snus",
    "alliance_abbr": "HOT",
    "created_at": "2026-06-15T05:30:45Z"
  }
]
```

| 필드         | 타입              | 설명                                       |
| ------------ | ----------------- | ------------------------------------------ |
| `id`         | number            | 뉴스 항목 ID                               |
| `kind`       | string            | 이벤트 종류 (`name_change`, `transfer` 등) |
| `message`    | string            | 사람이 읽기 쉬운 이벤트 메시지             |
| `player_id`  | string            | 해당 플레이어 ID                           |
| `old_name`   | string            | 변경 전 이름 (name_change 시)              |
| `state`      | number            | 왕국 번호                                  |
| `created_at` | string (ISO 8601) | 이벤트 발생 시각                           |

## 코드 예시

### JavaScript (fetch)

```javascript
const BASE_URL = "https://kingshot.jeab.dev";

// 태그된 플레이어 목록 가져오기
async function getTaggedPlayers(limit = 100, offset = 0) {
  const res = await fetch(
    `${BASE_URL}/api/players?tagged=1&limit=${limit}&offset=${offset}`,
  );
  return res.json(); // { players: [...], total: number }
}

// 플레이어 상세 정보
async function getPlayerDetail(playerId) {
  const res = await fetch(`${BASE_URL}/api/players/${playerId}`);
  return res.json();
}

// 플레이어 검색
async function searchPlayers(query, limit = 20) {
  const res = await fetch(
    `${BASE_URL}/api/players?search=${encodeURIComponent(query)}&limit=${limit}&offset=0`,
  );
  return res.json();
}

// 활성 기프트 코드만 필터
async function getActiveCodes() {
  const res = await fetch(`${BASE_URL}/api/codes`);
  const codes = await res.json();
  return codes.filter((c) => c.IsActive);
}
```

### Python (requests)

```python
import requests

BASE_URL = 'https://kingshot.jeab.dev'

def get_tagged_players(limit=100, offset=0):
    r = requests.get(f'{BASE_URL}/api/players', params={
        'tagged': 1, 'limit': limit, 'offset': offset
    })
    return r.json()  # { 'players': [...], 'total': int }

def get_player_detail(player_id: str):
    r = requests.get(f'{BASE_URL}/api/players/{player_id}')
    return r.json()

def search_players(query: str, limit=20):
    r = requests.get(f'{BASE_URL}/api/players', params={
        'search': query, 'limit': limit, 'offset': 0
    })
    return r.json()

def get_active_codes():
    r = requests.get(f'{BASE_URL}/api/codes')
    return [c for c in r.json() if c['IsActive']]
```

## 주요 참고 사항

- **공식 API가 아닙니다.** 개인이 운영하는 커뮤니티 트래킹 사이트이며, 스펙이 예고 없이 변경될 수 있습니다.
- `last_refreshed_at`으로 데이터 신선도를 확인하세요. 실시간이 아닌 주기적 수집 방식입니다.
- `sort_order`는 음수값이며, 절댓값이 클수록 상위 순위입니다 (전투력 기반 추정).
- 페이지네이션은 `limit` + `offset` 조합으로 구현하며, `total` 필드로 전체 수를 확인할 수 있습니다.
- Rate limit 정보는 확인되지 않았으므로, 대량 요청 시 딜레이를 두는 것을 권장합니다.
